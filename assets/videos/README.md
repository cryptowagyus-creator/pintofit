# Video Files — PintoFit

Drop your video files here. Name them exactly as shown below, then update
the matching `video:` field in `src/data/workouts.js`.

## How to wire a video

In workouts.js, change:
  video: null

To:
  video: require('../../assets/videos/incline_db_bench.mp4')
